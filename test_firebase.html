🔥 Diagnostic Firebase — TRANSCHAINES

📋 Si l'écriture est bloquée — Corriger les règles Firebase

  1. Aller sur console.firebase.google.com
  2. Cliquer sur votre projet erp-transchaines
  3. Menu gauche → Realtime Database
  4. Onglet Règles (en haut)
  5. Remplacer le contenu par:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}

6. Cliquer "Publier" — La synchronisation fonctionnera immédiatement.
